# starter-element

A simple demo starter element.

## Properties

| Property   | Attribute  | Type      | Default | Description                           |
|------------|------------|-----------|---------|---------------------------------------|
| `disabled` | `disabled` | `Boolean` | "false" | If true, the button will be disabled. |
| `role`     | `role`     | `String`  | "null"  | The ARIA role of the element.         |
